
# Nandini Chakral Portfolio Website

## Files Included
- index.html
- style.css
- script.js

## Features
- Modern responsive UI
- AI/Developer portfolio layout
- Animated sections
- Skills, Projects, Education, Contact
- Dynamic footer year script
- Glassmorphism effects
- Gradient graphics background

## How to Run
1. Extract the ZIP file
2. Open `index.html` in your browser

## Customize
- Replace text in index.html
- Update colors in style.css
- Add more animations in script.js
